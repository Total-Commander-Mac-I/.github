## 💡 Overview

<a href="#totalCommander">Total Commander</a> is the professional dual-pane file manager for Mac — bringing the two-panel Orthodox File Manager paradigm that power users have depended on for decades to macOS, replacing Finder's single-window approach with an efficient side-by-side navigation model built for keyboard-driven file operations.

The <a href="#totalCommander">dual-pane interface</a> shows two directory locations simultaneously — the source in one panel, the destination in the other — making copy, move, and comparison operations between two locations a matter of selecting files in one panel and executing the operation to the other without navigating away from either location. This paradigm eliminates the constant back-and-forth navigation that single-window file managers require for any operation involving two locations, and the efficiency difference compounds across the hundreds of file operations in a typical working day. The <a href="#totalCommander">keyboard-driven navigation</a> keeps all file operations accessible through keyboard shortcuts — navigating directories, selecting files, executing copy and move operations, opening files, and invoking tools without touching the trackpad. Users who invest in the keyboard shortcuts operate at speeds that mouse-driven file management cannot approach.

The <a href="#totalCommander">built-in file viewer</a> previews text files, images, hex content, and other formats in a popup window without opening the file in an external application — quickly checking file contents during a file management operation without the application launch time and window switching that external preview requires. The <a href="#totalCommander">FTP and SFTP client integration</a> connects to remote servers directly from Total Commander — navigating remote servers in one panel while the local filesystem is in the other, and transferring files between local and remote locations with the same copy and move operations used for local transfers.</p>

<a href="#totalCommander">Archive handling</a> opens ZIP, RAR, 7Z, and other archive formats as if they were directories — navigating into archives without extraction, extracting selected files rather than entire archives, and creating archives from selected files directly from the file manager interface. <a href="#totalCommander">File synchronization</a> compares two directories and synchronizes their contents — identifying files that exist in one location but not the other, files that differ between locations, and executing the synchronization operations to bring both locations into alignment.

<a href="#totalCommander">Batch rename</a> renames multiple files simultaneously with pattern-based rename rules — adding prefixes or suffixes, replacing text across many filenames, numbering files in sequence, changing case — the file renaming operations that photographers, developers, and media managers need to organize large file collections. <a href="#totalCommander">Advanced search</a> finds files by name, content, date, size, and other attributes across the entire filesystem or within specific directories — with regular expression support for pattern-based file finding.</p>

## 📌 FAQ

<details>
<summary>What is a dual-pane file manager?</summary>

A dual-pane file manager shows two directory locations simultaneously — source in one panel, destination in the other — enabling copy, move, and comparison operations between two locations without navigating away from either.

</details>

<details>
<summary>Does Total Commander support FTP and SFTP?</summary>

Yes. Total Commander includes FTP and SFTP client functionality for connecting to remote servers directly from the file manager.

</details>

<details>
<summary>Can Total Commander open archive files?</summary>

Yes. ZIP, RAR, 7Z, and other archive formats open as if they were directories — navigate and extract without leaving Total Commander.

</details>

<details>
<summary>Does Total Commander replace Finder completely?</summary>

Total Commander handles file management operations more efficiently than Finder for power users. Some macOS operations still require Finder or other system interfaces.

</details>

<details>
<summary>What is file synchronization in Total Commander?</summary>

File synchronization compares two directory locations, identifies differences — files present in one but not the other, files that differ — and executes operations to bring both locations into alignment.

</details>

<details>
<summary>Does Total Commander support keyboard shortcuts?</summary>

Yes. All file operations are accessible through keyboard shortcuts for mouse-free file management.

</details>

<details>
<summary>Can I batch rename files in Total Commander?</summary>

Yes. Pattern-based batch rename applies to multiple files simultaneously with prefix, suffix, numbering, case, and text replacement operations.

</details>

<details>
<summary>Does Total Commander run in dark mode?</summary>

Yes. Total Commander follows the macOS system appearance setting.

</details>

<details>

Yes. Current versions support Apple Silicon Macs.

</details>
